# Diabetes-Risk-Prediction-Project
Explainable & Fair Machine Learning for Diabetes Risk Prediction

📌 Overview
This project develops machine learning models to predict Type 2 Diabetes while ensuring interpretability (XAI) and fairness across demographic groups.
It was completed as part of my MSc in Artificial Intelligence.

🎯 Objectives
Build accurate diabetes prediction models
Apply Explainable AI (SHAP, LIME)
Evaluate fairness and bias
Balance performance, transparency, and fairness

📊 Data
Pima Indians Diabetes Dataset
NHANES dataset

⚙️ Methods
Models: Logistic Regression, Random Forest, XGBoost, Neural Networks
Explainability: SHAP, LIME
Fairness: Demographic Parity, Equal Opportunity

📈 Key Insights
Ensemble models performed best
Important features: Glucose, BMI, Age
Detected fairness gaps across demographic groups
