# Diabetes-Risk-Prediction-Project

Explainable & Fair ML for Diabetes Risk Prediction

📌 Overview

This project focuses on developing machine learning models for predicting Type 2 Diabetes while ensuring the models are both interpretable and fair. It integrates Explainable AI techniques and fairness evaluation to produce results that are not only accurate but also transparent and trustworthy. The work was completed as part of my MSc in Artificial Intelligence.

🎯 Objective

The aim of this project is to build reliable predictive models, apply explainability methods such as SHAP and LIME, and assess fairness across different demographic groups, ensuring a balanced approach between performance, interpretability, and ethical considerations.

📊 Data

The study makes use of the Pima Indians Diabetes dataset and the NHANES dataset, both of which provide relevant clinical and demographic features for diabetes risk prediction.

⚙️ Methodology

Several machine learning models, including Logistic Regression, Random Forest, XGBoost, and Neural Networks, were implemented and evaluated. Explainability techniques were applied to interpret model predictions, while fairness metrics were used to assess potential bias across groups.

📈 Results

The results show that ensemble models achieved strong predictive performance, with key features such as glucose level, BMI, and age playing significant roles. The analysis also highlights the presence of fairness disparities, emphasizing the importance of responsible AI in healthcare applications.

⚠️ Note

This project is intended for research purposes only and is not designed for direct clinical use.
